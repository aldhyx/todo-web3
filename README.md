## Web3 Todo App

This project is build for learning purpose using Foundry, Reactjs, Etherjs & vite.

## Documentation

- [Foundry](https://book.getfoundry.sh/)
- ReactJs
- Vite
- Etherjs

## Usage

### Build

```shell
$ make build
```

### Test

```shell
$ make test
```

### Format

```shell
$ make fmt
```

### Gas Snapshots

```shell
$ make snapshot
```

### Anvil

```shell
$ make anvil
```

### Deploy

```shell
$ make deploy
```

### Cast

```shell
$ cast <subcommand>
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
